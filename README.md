# 🏦 Bank Loan Risk Analysis Dashboard

## 📌 Project Overview
The **Bank Loan Risk Analysis Dashboard** is an end-to-end data analytics project designed to analyze loan applications, identify risky borrowers, and evaluate loan repayment performance.

This project uses **MySQL** for data cleaning, transformation, and advanced analysis, while **Power BI** is used to build an interactive dashboard for visual insights.

The goal is to help financial institutions improve loan approval decisions, reduce defaults, and optimize lending strategies.

---

## 🎯 Business Problem
Banks handle thousands of loan applications, but not all borrowers successfully repay their loans. Identifying risky customers before approval is crucial.

This project solves:

- Identifying high-risk loan applicants
- Understanding default patterns
- Improving loan approval decisions
- Monitoring repayment performance
- Reducing financial losses

## 📈 Key Insights

- Most loan defaults are concentrated in specific loan purposes.
- Higher Debt-to-Income ratio borrowers show higher risk.
- Customers with longer employment history have better repayment records.
- Certain states have higher loan approval and default rates.
- Good loans contribute the majority of recovered amounts.

---

## 💡 Business Recommendations

- Strengthen approval criteria for high DTI borrowers.
- Apply risk-based pricing for high-risk applicants.
- Improve monitoring for early payment defaults.
- Focus on low-risk customer segments for growth.
